function `gl_VertexID`
----------------------

.. currentmodule:: OpenGL.GLSL

Contains the index of the current vertex

.. c:function:: gl_VertexID


.. rubric:: Specification

.. c:function:: in int gl_VertexID)

.. seealso::

   
   :c:func:`gl_InstanceID`
   
.. rubric:: Description


:c:func:`gl_VertexID` is a vertex language input variable that holds an integer index for the vertex. The index is impliclty generated by :c:func:`glDrawArrays` and other commands that do not reference the content of the ``GL_ELEMENT_ARRAY_BUFFER``, or explicitly generated from the content of the ``GL_ELEMENT_ARRAY_BUFFER`` by commands such as :c:func:`glDrawElements`.
