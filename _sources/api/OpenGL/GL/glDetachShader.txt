function `glDetachShader`
-------------------------

.. source file man4/glDetachShader

.. currentmodule:: OpenGL.GL

Detaches a shader object from a program object to which it is attached

.. c:function:: glDetachShader
.. c:function:: void glDetachShader(GLuint  program, GLuint  shader)
   :noindex: 


   
   :param program:
      Specifies the program object from which to detach the shader object.
   
   :param shader:
      Specifies the shader object to be detached.
   

.. seealso::

   
   :c:func:`glAttachShader`
   
.. rubric:: Description


:c:func:`glDetachShader` detaches the shader object specified by *shader* from the program object specified by *program* . This command can be used to undo the effect of the command :c:func:`glAttachShader`.

If *shader* has already been flagged for deletion by a call to :c:func:`glDeleteShader` and it is not attached to any other program object, it will be deleted after it has been detached.


**Associated gets:**


   * :c:func:`glGetAttachedShaders` with the handle of a valid program object
   * :c:func:`glGetShader` with arguments *shader* and ``GL_DELETE_STATUS``
   * :c:func:`glIsProgram`
   * :c:func:`glIsShader`

**Errors:**

   * ``GL_INVALID_VALUE`` is generated if either *program* or *shader* is a value that was not generated by OpenGL.
   * ``GL_INVALID_OPERATION`` is generated if *program* is not a program object.
   * ``GL_INVALID_OPERATION`` is generated if *shader* is not a shader object.
   * ``GL_INVALID_OPERATION`` is generated if *shader* is not attached to *program* .
